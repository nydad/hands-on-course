# DECISIONS

판단 이력 — 번복 방지, 맥락 보존.

## 2026-04-13 — Setup

### D1: 루프 작업 타겟을 presentation.html 한 파일로 제한
- **이유**: tips.html은 독립 부록, 다른 실습 파일은 스코프 외. 집중도 확보.
- **영향**: 다른 파일 변경 시 명시적 재판단 필요.

### D2: 기존 팩트·인용 보존
- **이유**: 사용자 명시 — "내용을 전체적으로 왜곡하면 안 된다"
- **영향**: Jensen Huang, Andrew Ng, Addy Osmani, DORA, AlphaFold 등 수치·출처 절대 변경 금지.

### D3: 색상 팔레트는 기존 테마(indigo/rose/mint/cyan/amber) 유지
- **이유**: 전체 톤 일관성, 급격한 변화 금지
- **영향**: 새 색상 추가 대신 기존 색상의 opacity/weight 조정으로 가시성 확보.

### D4: 125% 스케일은 CSS `html { font-size: 117% }` 또는 clamp 상향 방식으로 적용
- **이유**: 레이아웃 파괴 최소화, 반응형 유지
- **영향**: vw 기반 폰트는 자동 확장. 고정 rem만 상향 조정.

### D5: Chrome QA는 GitHub Pages 배포본이 아닌 로컬 file:// 또는 실시간 반영 확인
- **이유**: 배포 지연·캐시 이슈 회피
- **영향**: 변경 후 즉시 시각 검증 가능.
